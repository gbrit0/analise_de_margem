# Análise de Margem

O projeto se divide em 3 etapas:

1 - [Automação da planilha](excel/): emissão automática a partir do banco de dados do Protheus de uma planilha que se assemelhe ao processo manual anterior realizado pela controladoria;
2 - Implementação Web da planilha: plataforma web que permita o registro histórico e acompanhamento das margens das empresas: BRG, Grid GO, Grid MG e Grid PA bem como o processo de análise em si;
3 - Análise automática: Adoção de algoritmo de inteligência artificial para sugestão de justificativas para margens fora do intervalo esperado.

Cada etapa será implementada individualmente porém servido de base para as etapas seguintes.
